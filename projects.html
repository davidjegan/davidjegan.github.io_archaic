Projects


Some certificates on leading domains, frameworks and technologies!!

Zappa based Alexa Chatbot

Amazon Alexa is a cloud-based voice service which interacts with users in an intuitive hands-free manner. Zappa is used to build a WSGI compstible Flask application which works with Python 2.7 & 3.6.

Serverless Slack Application

The repository demonstrates how to use AWS Cloudformation to build a serverless processing environment of AWS Lambda, Amazon API Gateway, AWS IAM and Amazon Rekognition. This workflow processes photos sent to Slack, uploaded to Amazon S3 and then extracts metadata from the image.

Expose Cassandra Data

The repository demonstrates how to expose data present in the Cassandra DB running in an instance to external world by running a NodeJS server.

AWS EMR Node Calculator

In order to ensure parallelism, perfect number of nodes should be chosen in EMR Clusters. This involves a complex look up and referencing. Using this tool, that arduous process is simplified. This tool, returns the exact nodes required for your application to run seemlessly. This repo helps to identify the perfect node.

AWS Instance Status Automation

A serverless way of automating the Amazon web services EC2 instance running pattern. Has lambda in the background and is triggered by Cloudwatch cron jobs!
